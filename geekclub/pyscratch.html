hsv_to_hex(h, s, v)
Convert HSV values (0-1) to a hex string #000000-#ffffff.
 
The first number is hue: red=0, green=0.333
The second saturation (how much colour)
And last brightness: black=0, brightest=1
 
White:
>>> hsv_to_hex(0, 0, 1)
'#ffffff'
 
Red:
>>> hsv_to_hex(0, 1, 1)
'#ff0000'
